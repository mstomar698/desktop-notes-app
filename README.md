### Notes Web App

> Uses Tauri, Next JS, Kuzu DB

> Built with Rust, TypeScript, Tailwind CSS

### Development
```bash
### clone the repo
git clone git remote add origin https://github.com/mstomar698/desktop-notes-app.git -b main

### install dependencies
npm install

### run the app
npm run tauri dev
```

### Build
```bash
### build frontend
npm run build
### build the app
npm run tauri build
```
